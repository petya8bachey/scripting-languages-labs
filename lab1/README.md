# Лабораторная работа №1

## Задание №1.18

Реализовать алгоритм сжатия RLE (Run-Length Encoding).
Реализация заключается в создании двух функций: `compress_rle(data)`, которая заменяет последовательности повторяющихся символов на пары (символ, количество), и `decompress_rle(data)`, восстанавливающей исходную строку.
