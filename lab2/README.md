# Лабораторная работа №2

## Задание №18

### Клиенты из двух баз

Даны два файла: `customers1.txt` и `customers2.txt`, в каждом — список email-адресов клиентов (по одному на строке).
Прочитайте адреса в два множества. Найдите и выведите:
 - клиентов, которые есть в первой базе, но отсутствуют во второй;
 - клиентов, присутствующих в обеих базах;
 - объединённый список всех клиентов.
