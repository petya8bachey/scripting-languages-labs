# Лабораторная работа №5

## Работа с базами данных (PostgreSQL)

Спроектировать и реализовать базу данных с 2–3 таблицами, связанными отношениями «один к одному», «один ко многим» и «многие ко многим». Требуется разработать программный модуль на Python для выполнения операций создания, чтения, обновления и удаления данных (CRUD) с использованием как прямых SQL-запросов, так и ORM-абстракции SQLAlchemy. Также необходимо реализовать фильтрацию записей, управление транзакциями и настройку подключений к базе данных.
